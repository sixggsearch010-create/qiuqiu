# 小球吞噬战

一个浏览器端小球吞噬游戏，支持单人游玩和局域网多人联机。玩家通过吞食、分裂、吐球、加速和聚合来成长，并与 AI 或局域网玩家争夺榜首。

## 功能特点

- 单人模式：直接在浏览器中打开 `小游戏.html` 即可游玩。
- 局域网多人：使用 `server.js` 启动 WebSocket 中继服务器，支持最多 4 人连接。
- 双操作模式：支持鼠标操作和键盘操作。
- 成长与战斗：包含分裂追击、吐球爆炸、加速、聚合、AI 对手、排行榜等玩法。

## 环境要求

- Node.js 16 或更高版本
- 现代浏览器，例如 Chrome、Edge、Firefox

## 快速开始

安装依赖：

```bash
npm install
```

启动联机服务器：

```bash
npm start
```

默认端口是 `8080`。如果需要指定端口，可以运行：

```bash
npm start -- 3000
```

启动后终端会输出类似下面的地址：

```text
ws://192.168.x.x:8080
ws://localhost:8080
```

## 游玩方式

### 单人模式

直接用浏览器打开 `小游戏.html`，点击“开始游戏”。

### 局域网多人

1. 房主先运行 `npm start` 启动服务器。
2. 所有玩家使用浏览器打开 `小游戏.html`。
3. 房主点击“局域网多人” -> “创建房间 (主机)”。
4. 其他玩家点击“局域网多人” -> “加入房间”，输入房主终端显示的 `主机IP:端口`，例如 `192.168.1.5:8080`。
5. 房主在大厅点击“开始游戏”。

## 操作说明

鼠标模式：

- 鼠标指向移动
- 左键分裂
- 右键吐球
- Shift 加速
- F 聚合

键盘模式：

- WASD / 方向键移动
- Space 分裂
- E / K 吐球
- Shift 加速
- F 聚合
